# Nexus Arcade — AngularJS

Protótipo de uma plataforma gamer desenvolvido em AngularJS, utilizando módulo principal, controlador, serviços e diretivas customizadas.

## Componentes implementados como diretivas

- `accordion`
- `pagination`
- `progress-bar`
- `tabs`
- `dropdown-menu`
- `toast-notification`

## Executar

Abra `index.html` diretamente no navegador ou use:

```bash
python -m http.server 8000
```

Depois acesse `http://localhost:8000`.
