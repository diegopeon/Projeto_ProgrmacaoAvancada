# Nexus Arcade — Vue + Vite

Protótipo de uma plataforma gamer desenvolvido em Vue.js com Vite.js.

## Componentes implementados

- `Accordion.vue`
- `Pagination.vue`
- `ProgressBar.vue`
- `Tabs.vue`
- `DropdownMenu.vue`
- `ToastNotification.vue`

## Executar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Conceitos Vue demonstrados

- Componentes reutilizáveis
- Props
- Emits
- Estado local com `ref`
- Computed properties
- Lifecycle hook `onMounted`
- Event handling com `@click`
